PHP漏洞靶场使用说明

项目简介

本项目是一个用于测试防火墙漏洞防护能力的PHP靶场，包含以下漏洞：

  • SQL注入（登录/注册页面）
  • 命令注入（Ping功能）
  • XSS（留言板）
  • 文件包含（文件读取功能）

安全警告

本靶场仅用于测试目的，请勿在生产环境部署。包含多个已知漏洞，用于测试防火墙防护能力。

访问地址

本地访问：http://localhost:3009

初始账号

管理员账号：admin

密码：guardrails

功能说明

  • 登录/注册：测试SQL注入漏洞
  • 网络测速面板：登录后首页，包含所有功能导航
  • Ping功能：测试命令注入漏洞
  • 留言板：测试XSS漏洞
  • 文件读取：测试文件包含漏洞
  • 管理员编辑：仅管理员可访问，用于修改文档
